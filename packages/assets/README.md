# @kata-kit/assets

> kata-kit asset package.

## Install

```sh
# yarn
yarn add @kata-kit/assets
# npm
npm install @kata-kit/assets
```

## Usage

Simply import the images you'd like, e.g.:

```javascript
import '@kata-kit/assets/images/form-empty.svg';
```
