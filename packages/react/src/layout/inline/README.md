# Inline

Use the `<Inline />` component to evenly space inner elements horizontally.

## Usage

```jsx
import { Inline } from '@aksara-ui/react';

export default function Component({ children }) {
  return (
    <Inline>
      <Box>example</Box>
      <Box>example</Box>
      <Box>example</Box>
    </Inline>
  );
}
```
